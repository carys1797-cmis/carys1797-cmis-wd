Creating a Webpage

  1. Create two new files in gedit, one called "index.html" and the other called "main.css"
  2. Open up template.html in gedit and copy and paste the contents into your new "index" file
  3. Add text to your index file by typing in between the closing and opening tags of each element
  4. To edit the style and colours of your webpage by editing your main.css file, type the name of the element of which you want to edit and then "{}". In between the curly brackets type what aspect of the element you want to change, and then what you wantto change it to
  5. To view your html file while you are editing it, you can type the command "python -m SimpleHTTPServer" into terminal and then in a new tab (on the internet) type "localhost:8000"
  6. Don't forget to validate frequently
  7. When your webpage looks the way you want it to, validate both files and save them
  8. Then add, commit and push them back to your github account
  9. To upload them to the production server, open up sftp in terminal by typing the command "sftp (username)@cmis-hsteacher". Make sure you are in your repository folder before you open sftp
  10. When a yes/no question pops up, type yes
  11. Enter the public html folder by typing the command "cd public_html" into terminal
  12. To upload files to the production server, type "put" and then the name of the file you want to upload
  13. To view the files you have type "ls" into terminal
  14. Finally to exit sftp, type "exit", and hit enter
